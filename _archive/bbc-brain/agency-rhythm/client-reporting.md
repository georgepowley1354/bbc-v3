# Client Reporting

Templates for keeping clients informed without over-communicating.

---

## Status Update Template (Active Project)

Send on Fridays for any project with a client waiting on something.

```
Subject: [Client Name] — Site Update [Date]

Hi [Name],

Quick update on where we are:

✓ Done this week: [1-2 things completed]
→ In progress: [what's actively being worked on]
→ Next: [what's coming next week]

[If waiting on client:] I need [X] from you to move forward with [Y]. Can you get that to me by [date]?

[If no blockers:] Nothing needed from you right now — I'll follow up when [next milestone] is ready for your review.

— Jorge
```

## Design Review Request Template

When design is ready for client feedback:

```
Subject: [Client Name] — Design Ready for Review

Hi [Name],

Your [page/design] is ready for review.

View it here: [Figma link or staging URL]

Please review and share feedback by [date]. A few things to focus on:
- Does this feel right for your brand?
- Is anything missing from this page?
- Any copy changes needed?

This is our [1st/2nd] revision round. [We have 1 more included in the scope / Additional revisions are $X/round.]

— Jorge
```

## Monthly Retainer Report Template

Send on the 1st of each month for all Plan B and Plan C retainer clients.

```
Subject: [Client Name] — Monthly Site Report — [Month Year]

Hi [Name],

Here's your monthly report for [domain]:

**Performance**
- Visits: [X] ([+/-X%] vs. last month)
- Top pages: [page 1], [page 2], [page 3]
- Core Web Vitals: [Pass/Fail — link to report]

**What We Did This Month**
- [Update 1]
- [Update 2]
- [Maintenance: dependencies updated, security patches applied]

**SEO**
- [1-2 observations — ranking improvements, new keywords, etc.]

**Recommended Next Steps**
- [1-2 specific suggestions — new page, content update, etc.]

Hours used this month: [X of Y included]

Let me know if you want to talk through any of this.

— Jorge
```

## Communication Rules

1. **One update per week max** for active projects (Friday). Don't send mid-week updates unless something urgent changed.
2. **Respond to client messages within 24 hours on business days.** Don't let messages sit.
3. **Never send bad news by surprise.** If a timeline is slipping, tell the client before the deadline, not after.
4. **Keep updates short.** Clients don't read long emails. 3-5 lines max.
5. **Be specific about what you need from them.** "Let me know your thoughts" gets vague replies. "I need your approved logo and menu PDF by Thursday" gets results.
