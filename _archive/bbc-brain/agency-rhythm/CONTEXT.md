## The Rhythm in One View

```
Monday:     Review project status. Set weekly priorities. Clear inbox.
Tue–Thu:    Heads-down build work. Client communication between 9–11am only.
Friday:     Send client updates. Review the week. Update project-bank.md.
Monthly:    Send retainer reports. Review pipeline in project-engine.
Quarterly:  Review service pricing. Archive closed projects.
```
