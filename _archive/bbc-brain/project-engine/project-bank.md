## How to Add a New Project

1. Use the next P-ID (check last row in Active section)
2. Fill in Client, Status = `lead` or `discovery`
3. Estimate tier: T1 (starter), T2 (custom), T3 (retainer-based)
4. Note which service pillars (S1-S6) apply
5. Add notes on source (referral? cold inquiry? existing client?)
