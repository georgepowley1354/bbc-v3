Create a temporary Node.js script at `bbc-v3/generate-assets.js` that uses the sharp library (already in devDependencies) to generate all 13 image files. Then run it and delete it.

**Step 1: Ensure directories exist**
```bash
mkdir -p bbc-v3/images/icons
```

**Step 2: Create `bbc-v3/generate-assets.js`** with the following logic:

**OG Image (1200x630 JPEG):**
- Create an SVG at 1200x630 with background `#0E0E0E`
- Top accent bar: rect full-width, 4px tall at y=0, fill `#FF4D1C`
- "BIG BAD CODING" text: centered at y=290, font-size 80px, font-weight 900, font-family "Arial Black, Impact, sans-serif", fill `#F0EDE8`, letter-spacing 8px
- Tagline "Hand-Coded Websites for Capital Region NY" centered at y=350, font-size 26px, font-family "Arial, Helvetica, sans-serif", fill `#9E9891`
- URL "bigbadcoding.com" centered at y=570, font-size 22px, font-family "Courier New, monospace", fill `#FF4D1C`
- Bottom accent bar: rect full-width, 4px tall at y=626, fill `#FF4D1C`
- Use sharp to create a raw #0E0E0E background, composite the SVG overlay, output as JPEG quality 90 to `bbc-v3/images/og-image.jpg`

**PWA Icon (512x512 master, resized to 8 sizes):**
- Create an SVG at 512x512 with background `#0E0E0E`
- Centered "BBC" text: font-size 200px, font-weight 900, font-family "Arial Black, Impact, sans-serif", fill `#FF4D1C`, text-anchor middle, dominant-baseline central, x=256, y=256
- Use sharp to render the SVG to a 512x512 PNG
- Resize that master to each manifest size: 72, 96, 128, 144, 152, 192, 384, 512
- Save each as `bbc-v3/images/icons/icon-{size}x{size}.png`

**Apple Touch Icons (master at 180px, resized to 4 sizes):**
- Same design as PWA icon but with a subtle rounded-rect background: full 180x180 `#0E0E0E` background with "BBC" text scaled proportionally (font-size ~70px)
- Create a 180x180 master, resize to 120, 152, 167, 180
- Save each as `bbc-v3/images/icons/apple-touch-icon-{size}x{size}.png`

**Implementation notes for the script:**
- Use `sharp({ create: { width, height, channels: 4, background: { r: 14, g: 14, b: 14, alpha: 1 } } })` to create base images
- Use `.composite([{ input: Buffer.from(svgString), top: 0, left: 0 }])` to overlay SVG text
- For resizing: create the largest size first, then use `sharp(masterBuffer).resize(size, size).png().toBuffer()` for each smaller size
- All operations are async — use `await` with `sharp().toBuffer()` and `sharp().toFile()`

**Step 3: Run the script**
```bash
cd bbc-v3 && node generate-assets.js
```

**Step 4: Delete the generator script**
```bash
rm bbc-v3/generate-assets.js
```

ls -la bbc-v3/images/og-image.jpg bbc-v3/images/icons/icon-*.png bbc-v3/images/icons/apple-touch-icon-*.png
